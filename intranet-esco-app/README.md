# ESCOFER Intranet (Frontend)

## Requisitos
- Node.js 18+
- npm

## Rodar localmente
```sh
npm install
npm run dev
```

Abra http://localhost:8080 no navegador.

## Scripts
- `npm run dev`: servidor de desenvolvimento
- `npm run build`: build de producao
- `npm run preview`: preview do build
- `npm run lint`: lint do projeto

## Stack
- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
